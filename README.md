# VAProxy.FreecamMod

Simple freecam mod for VA Proxy.

Shift to 4x speed, ctrl/pgdn to go down, space/pgup to go up, wasd for movement.

Scroll up to increase speed, scroll down to decrease speed.

F9 to toggle UI.

F10 to toggle info ui.
